# hash

This project was created using `bun init` in bun v1.0.4. [Bun](https://bun.sh) is a fast all-in-one JavaScript runtime.

Run this cmd to install bun:

```bash
curl -fsSL https://bun.sh/install | bash
```

To install dependencies:

```bash
bun install
```

To run tests:

```bash
bun run:test
```

To run:

```bash
bun run:start
```
